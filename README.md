# Pandira

Pandira es una aplicación web personal para gestionar colecciones de entretenimiento.

Permite organizar y visualizar tu colección de:

- 🎬 Películas
- 🎮 Videojuegos
- 📁 Colecciones personalizadas

La aplicación obtiene automáticamente información y portadas desde APIs externas para enriquecer tu colección.

---

# Tecnologías

Pandira está construida con:

- **Next.js (App Router)**
- **Supabase** (base de datos y autenticación)
- **TailwindCSS**
- **Vercel** (deploy)

---

# Funcionalidades actuales

- Login de usuario
- Gestión de colección de películas
- Gestión de colección de videojuegos
- Enriquecimiento automático de datos
- Dashboard de estadísticas
- Sistema de colecciones personalizadas
